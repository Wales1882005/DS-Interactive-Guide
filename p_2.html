Tree Anatomy & Binary Trees

This section breaks down the hierarchical nature of Tree Data Structures. You will explore the terminology used to describe tree components and understand the specific constraint of a Binary Tree, where each node can have a maximum of two children.

Binary Tree Structure

Terminology Glossary

  • Root The topmost node of the tree. There is only one root per tree.
  • Parent & Child A node with an edge downward is a parent. The node below is its child.
  • Leaf A node which does not have any child nodes (the bottom of the branches).
  • Subtree Represents the descendants of a node, forming a smaller tree within.
